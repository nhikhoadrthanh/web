# Hướng dẫn chạy Website Nhi Khoa DrThanh

Website được xây dựng bằng **HTML, CSS và JavaScript thuần**, vì vậy bạn **không cần cài đặt** bất kỳ phần mềm phức tạp nào (như Node.js).

## 1. Cách chạy website
Cách đơn giản nhất:
1. Mở thư mục dự án `d:\Web DrThanh`.
2. Tìm file `index.html`.
3. **Click đúp chuột** vào file `index.html` để mở trên trình duyệt (Chrome, Cốc Cốc, Edge...).

## 2. Cấu trúc thư mục
- `index.html`: Trang chủ.
- `about.html`: Trang giới thiệu.
- `booking.html`: Trang đặt lịch khám.
- `consultation.html`: Trang tư vấn online.
- `blog.html`: Trang danh sách bài viết.
- `blog-detail.html`: Trang chi tiết bài viết mẫu.
- `resources.html`: Trang tải tài liệu miễn phí.
- `products.html`: Trang sản phẩm số.
- `css/style.css`: File chứa toàn bộ giao diện (màu sắc, phông chữ).
- `js/main.js`: File xử lý logic (menu mobile, form).
- `public/`: Thư mục chứa hình ảnh.

## 3. Chỉnh sửa
Bạn có thể mở bất kỳ file `.html`, `.css` nào bằng Notepad, VS Code hoặc bất kỳ trình soạn thảo văn bản nào để chỉnh sửa nội dung.

## 4. Lưu ý
- Form "Đặt lịch" và "Nhận tài liệu" hiện tại chỉ hiện thông báo (Alert) mô phỏng thành công. Để hoạt động thật, cần tích hợp Google Form hoặc Backend.
- Giao diện đã được tối ưu cho Mobile. Hãy thu nhỏ trình duyệt để kiểm tra.
